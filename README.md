# My Personal Website

This repository contains the source code for my personal website, built with
[Zola](https://getzola.org).

## Design goals

- Multilingual support (Spanish as default and English)
- Semantic HTML
- SEO-friendly

## Requirements

- Zola

## Development

Start the local server:

```bash
zola serve
```

Build the static site:

```bash
zola build
```
